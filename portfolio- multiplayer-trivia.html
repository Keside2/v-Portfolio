Project Overview

A real-time multiplayer trivia game built with React, TypeScript, and Firebase Realtime Database. Players can create or join rooms, compete in trivia rounds, and chat live. Fast answers earn bonus points!

Key Features

Responsive Design

The game is fully optimized for desktops, tablets, and mobile devices, providing a seamless and engaging experience across all screen sizes.

Real-Time Multiplayer Gameplay

Powered by Firebase Realtime Database, players can create or join game rooms and compete simultaneously with others. Game state updates instantly, ensuring a smooth and interactive multiplayer experience.

Performance Optimization

Built with React and TypeScript for efficient rendering and fast interactions. Optimized state management ensures minimal latency and smooth gameplay even during active sessions.

Live Chat System

Includes an integrated real-time chat feature that allows players within a room to communicate, enhancing engagement and social interaction during gameplay.

Dynamic Scoring System

Implements a time-based scoring mechanism where faster answers earn bonus points, increasing competitiveness and excitement.

Secure & Scalable Architecture

Utilizes Firebase’s secure backend services to handle user sessions, data synchronization, and scalability, ensuring reliable performance as user activity grows.
